## File 관련 유틸

### File
- directory 생성
- txt 읽기 쓰기
- 파일 이름 변경 및 삭제
- 해당 디렉토리 파일 읽기

### XML
- Jdom을 활용하여 xml 생성
- Jdom을 활용하여 xml 읽기

### Excel
- POI를 활용하여 excel 생성
- POI를 활용하여 excel 읽기

### QR Code
- zxing을 활용하여 QR코드 생성


### regularexpression
- resource의 regex_print.txt를 읽어서 졍규퓨현식으로 키워드 추출
- resource의regex_remove.txt를 읽어서 졍규퓨현식으로 키워드 제거 


### RestApi
- HTTPClient 사용하여 GET, POST방식으로 RESTAPI 호출
